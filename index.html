Технологии необходимые для создания сайтов


HTML, CSS, JavaScript и PHP

HTML (HyperText Markup Language) – язык гипертекстовой разметки документа. Основное его назначение – вывести на монитор текст, пригодный для чтения.

CSS (Cascading Style Sheets) – язык описания внешнего вида документа.

JavaScript – позволяет реагировать на действия пользователя моментально на этой же странице без её перезагрузки, обеспечивая интерактивность.

PHP (Hypertext Preprocessor, изначально Personal Home Page Tools) – это динамика сайта, его наполнение.
